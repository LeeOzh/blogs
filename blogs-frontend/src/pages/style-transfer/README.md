# AI图片风格转换功能

## 功能概述

这是一个基于AI的图片风格转换工具，用户可以上传图片并选择不同的艺术风格，让AI将图片转换为对应的艺术风格。

## 主要特性

### 🎨 多种艺术风格
- **动漫风格**: 日式动漫/漫画风格转换
- **油画风格**: 经典油画艺术效果
- **水彩画**: 清新的水彩画风格
- **赛博朋克**: 未来科技感风格
- **复古风格**: 怀旧复古视觉效果
- **素描风格**: 黑白素描艺术风格

### 📱 用户友好的界面
- **拖拽上传**: 支持点击和拖拽上传图片
- **实时预览**: 上传后立即显示图片预览
- **风格选择**: 直观的风格卡片选择界面
- **处理状态**: 清晰的处理进度指示
- **结果展示**: 优雅的结果展示和下载功能

### 🎯 交互体验
- **响应式设计**: 适配各种设备屏幕
- **深色模式**: 支持深色主题
- **动画效果**: 流畅的过渡动画
- **状态反馈**: 清晰的操作状态提示

## 技术架构

### 前端组件结构
```
style-transfer/
├── index.tsx          # 主组件
├── styles.css         # 自定义样式
└── README.md         # 功能说明
```

### 状态管理
- `selectedImage`: 用户选择的图片文件
- `previewUrl`: 图片预览URL
- `selectedStyle`: 选中的风格ID
- `isProcessing`: 处理状态
- `resultImage`: 转换结果图片

### 风格配置
每个风格包含以下属性：
- `id`: 唯一标识符
- `name`: 显示名称
- `description`: 风格描述
- `preview`: 预览图片路径
- `prompt`: AI提示词

## 使用流程

1. **上传图片**: 点击上传区域选择图片文件
2. **选择风格**: 从6种预设风格中选择一种
3. **开始转换**: 点击"开始转换"按钮
4. **等待处理**: AI处理过程中显示加载动画
5. **查看结果**: 处理完成后显示转换结果
6. **下载图片**: 可以下载转换后的图片
